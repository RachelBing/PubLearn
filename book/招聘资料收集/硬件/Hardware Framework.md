\*\*RISC-V \*\*

> RISC(精简指令集计算机，Reduced Instruction Set Computer-RISC)和CISC(复杂指令集计算机，Complex Instruction Set Computer-CISC)是CPU的两种架构，区别在于不同的CPU设计理念和方法。早期的CPU全部是CISC架构。
>
> VLSI（超 大规模集成电路）技术
>
> RISC-V读作RISC Five，也即第五代精简指令处理器。
